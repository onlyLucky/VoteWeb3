## web3.0 投票系统

### 基础结构

- [C 端](./client/)

### 需求分析

**C 端**

- 需要支持多种方式投票， 1.选举类型 2.投票决策方案类型 3.pk 支持双方哪一边类型（选举类型为多项）
- 用户可以，便捷快速进行各种类型创建投票，进行查看投票，进行投票。选择 web3.0 安全投票
- 可以进行关注某些投票，选择进行短信邮箱提醒。
- 便捷投票，选举类型。分享投票给指定某个人。
